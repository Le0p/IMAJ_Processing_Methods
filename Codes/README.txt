Important files:

- Color_library : Functions used for K-Means color quantization
- Wu_Color_library : Functions used for Wu color quantization
- Color_feature_extraction : To extract features with K-Means color quantization algorithm on a batch of images (possible to run it with the run.zsh script)
- Wu_Color_extraction : To extract features with Wu color quantization algorithm on a batch of images (possible to run it with the run.zsh script)
- Classification_pipeline : Code to run to run the classification pipeline
- Image_utils : util functions related to image loading and finding
- Classification_utils : util functions related to the classification pipeline
- Database_creation : Allow for the extraction and the creation of the database based on all images on a certain folder
- Dataset_creation : Allows for the creation of the dataset with a training list and a validation list
